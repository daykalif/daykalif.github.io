自学Java开发的学习路线

Posted on 2019-03-20 | In server | | Visitors:
Words count in article: 577 | Reading time ≈ 2

入坑 Java 开发的自学之路

基础知识

  • 编程语言：Java，Python， C
  • 基本算法
  • 基本网络知识：TCP/IP HTTP/HTTPS
  • 基本的设计模式

工具方面

  • 操作系统：Linux（CentOS/Ubuntu）
  • 代码管理：svn / git
  • 持续集成（CI/CD）：jenkins
  • Java 的项目管理工具：Maven / Gradle

框架

应用层框架

  • ssh: spring + structs + hibernate
  • ssm: spring springmvc(web 框架) mybatis（数据库的访问层）
  • spring boot

各种中间件

  • MQ 消息队列
  • RPC 通信框架 gRPC thrift dubbo springcloud
  • elasticsearch 数据库 搜索引擎

数据库

  • SQL：MySQL（关系型数据库） / Postgre SQL
  • NoSQL：(非关系型数据库) Redis Memcached mongodb elasticsearch

架构方面

分布式/微服务架构

  • spring cloud
  • dubbo
  • rpc 通信框架

虚拟化/容器化的技术

  • Docker 容器化技术
  • k8s kubernetes

关注源码/性能

  • JDK 源码以及部分设计思想
  • Spring 源码
  • JVM 细节与排错
  • 高并发/高可用

JAVA 书籍推荐

  • 《Java 编程思想》-语言
  • 《Java 并发编程实战》-多线程
  • 《深入理解 Java 虚拟机》-jvm
  • 《函数式编程思维》
  • 《TCP/IP》详解
  • 《Spring MVC+MyBatic 开发入门到项目实战》
    spring -> spring mvc -> spring boot -> spring cloud
  • 《Spring 技术内幕 深入解析 spring 架构与设计原理》
  • 《Elasticsearch 服务器开发》
  • 《Redis 入门指南》

数据结构和算法相关的书籍推荐

  • 《大话数据结构》-排序、查找、优化
  • 《算法导论》
  • 《剑指 offer》
  • 《编程之美》
  • 《编程珠玑》
  • LeetCode 题解

各种书籍

C 语言：

  • 《C Primer Plus》-指针、内存管理 - 第六版
  • 《C++ Primer Plus》
  • 《C++Primer》

Java：

  • 《Java 编程思想》
  • 《Java 核心技术》卷 1 卷 2

Python：

  • 《Python 核心编程》
  • 《Python 编程从入门到实战》

Go：
《Go 语言实战》 or 《Go In Action》

算法：

  • 《算法导论》-不适合初学者
  • 《算法》第四版 红色

计算机网络：【TCP/IP 协议栈】

  • 《TCP/IP 详解》
  • 《计算机网络 自顶向下》

Linux：

  • 《Linux 私房菜》-工具书
  • 《Unix 环境高级编程》

JavaEE 框架：【spring 框架】

  • 《Spring 实战》
  • 《Spring Boot 实战》
  • 《Spring 技术内幕》-很难

数据库：

  • 《MySQL 必知必会》 涵盖 《SQL 必知必会》的内容
  • 《高性能 MySQL》

代码质量：

  • 《重构 改善机构代码的设计》需要一些经验，拔高内容
