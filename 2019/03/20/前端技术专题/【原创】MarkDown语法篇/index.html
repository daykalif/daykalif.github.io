十一、字体颜色

使用简书的 markdown 不能改变字体颜色真是一件头疼的事，不过找了很久总算找到一种可以书写不同颜色的字的方法，语法如下：
$\color{red}{红色字}$

$\color{red}{红色字}$

当然表示成 rgb 十六进制或十进制形式都是可以的：
$\color{&#35;FF0000}{红色字}$或$\color{rgb(255,0,0)}{红色字}$
因此颜色可以灵活多变，具体颜色表可以参考链接

markdown 语法可以使用[TOC]来自动生成 markdown 文件的标题目录，但是 github 或者 gitee 却不支持[TOC]标签，解决如下： 1.下载 Visual Studio Code 2.打开 VScode，点击扩展，搜索 Markdown All in One 插件，安装 3.在 VScode 中打开 md 文件 4.使用 Ctrl+Shift+P 或者 command+shift+P 快捷键，输入命令 Markdown All in One: Create Table of Contents，回车
