开发环境

git

最常用的代码版本管理工具
大型项目需要多人协作开发，必须熟用 git

Mac OS 自带 git 命令，windows 可去官网下载安装
git 服务端常见的有 github coding.net 等
大公司会搭建自己的内网 git 服务

git status:查看改动
git diff/git diff 文件名 :直观查看修改到了哪些内容
git add 文件名/git add . ：提交所有的文件
git commit -m “xxx”：生成一条记录
git log:查看提交记录
git show xxxxxxxxxxxxxxxxxxxxxxx:查看本次提交的记录
git checkout 文件名/git checkout . :撤销
git push origin 分支名：将修改 push 到服务端
git pull origin master：将服务端最新代码拉下来
git branch：获取分支
git checkout -b 分支名/git checkout 分支名：拉取一个分支/切换分枝
git fetch:将服务端所有分支都拉下来
git merge 分支名：合并代码
git stash:不小心在主分支做了修改，可以先放到 stash 中，再切换回子分支
git stash pop:切换回子分支后，将 stash 中的内容放到子分支中

调试工具

Elements:Dom 结构的展示
Console：打印一些东西
Source:源码（debugger，打断点）
Network：资源的加载
Application：localStorage,sessionStorage 本地存储

（Chrome 还有包括比如：查看内存泄露，…）

抓包

移动端 h5 页，查看网络请求，需要抓包工具
windows 一般用 fiddler
Mac OS 一般用 charles

手机和电脑连同一个局域网
将手机代理到电脑上
手机浏览网页，即可抓包
查看网络请求
网址代理
https

webpack 和 babel
ES6 模块化，浏览器暂不支持
ES6 语法，浏览器并不完全支持
压缩代码，整合代码，以让网页加载更快

linux 常用命令

公司的线上机器一般都是 linux（参考阿里云）
测试机也需要保持一致，用 linux
测试机或者线上机出了问题，本地又不能复现，需要去排查

例如：
本地登陆到线上机器：ssh root@192.168.10.21 ,然后输入密码，就可以进入 linux 环境
查看文件夹：ls
查看文件列表：ll
清屏：clear
新建文件夹：mkdir abc
删除文件夹：rm -rf abc【-r:递归删除，-f:强制删除】
定位到目录：cd
修改文件名：mv index.html index1.html【将 index.js 文件名修改成 index1.html】
移动文件：mv bundle.js ../ 【将 bundle.js 移动到上级目录】
拷贝：cp a.js a1.js 【将 a.js 拷贝一份叫 a1.js】
删除文件：rm a1.js
新建文件：touch d.js
新建文件并打开：vi c.js 或者 vim c.js
查看文件：cat a.js 或者 head a.js【打印开头几行】 或者 tail a.js【打印末尾几行】 或者 grep “babel” package.js【查看文件中是否包含 babel】
学习 vim 命令：vimtutor

$\color{&#35;FF0000}{红色字}$或$\color{rgb(255,0,0)}{红色字}$

入坑 Java 开发的自学之路

基础知识

  • 编程语言：Java，Python， C
  • 基本算法
  • 基本网络知识：TCP/IP HTTP/HTTPS
  • 基本的设计模式

工具方面

  • 操作系统：Linux（CentOS/Ubuntu）
  • 代码管理：svn / git
  • 持续集成（CI/CD）：jenkins
  • Java 的项目管理工具：Maven / Gradle

框架

应用层框架

  • ssh: spring + structs + hibernate
  • ssm: spring springmvc(web 框架) mybatis（数据库的访问层）
  • spring boot

各种中间件

  • MQ 消息队列
  • RPC 通信框架 gRPC thrift dubbo springcloud
  • elasticsearch 数据库 搜索引擎

数据库

  • SQL：MySQL（关系型数据库） / Postgre SQL
  • NoSQL：(非关系型数据库) Redis Memcached mongodb elasticsearch

架构方面

分布式/微服务架构

  • spring cloud
  • dubbo
  • rpc 通信框架

虚拟化/容器化的技术

  • Docker 容器化技术
  • k8s kubernetes

关注源码/性能

  • JDK 源码以及部分设计思想
  • Spring 源码
  • JVM 细节与排错
  • 高并发/高可用

JAVA 书籍推荐

  • 《Java 编程思想》-语言
  • 《Java 并发编程实战》-多线程
  • 《深入理解 Java 虚拟机》-jvm
  • 《函数式编程思维》
  • 《TCP/IP》详解
  • 《Spring MVC+MyBatic 开发入门到项目实战》
    spring -> spring mvc -> spring boot -> spring cloud
  • 《Spring 技术内幕 深入解析 spring 架构与设计原理》
  • 《Elasticsearch 服务器开发》
  • 《Redis 入门指南》

数据结构和算法相关的书籍推荐

  • 《大话数据结构》-排序、查找、优化
  • 《算法导论》
  • 《剑指 offer》
  • 《编程之美》
  • 《编程珠玑》
  • LeetCode 题解
