JS Web API - DOM

一、DOM【DOM 操作（文档对象模型）】

Vue 和 React 框架应用广泛，封装了 DOM 操作
但 DOM 操作一直都会前端工程师的基础、必备知识
只会 Vue 而不懂 DOM 操作的前端程序员，不会长久

1.DOM 本质：【html 语言，或者 html 文件解析出来的一棵 DOM 树】

DOM 是哪种数据结构？—–树（DOM 树）

2.DOM 操作的常用 API：

1.DOM 节点操作

获取 DOM 节点

2.DOM 结构操作

新增/插入节点
获取子元素列表，获取父元素
删除子元素

3.DOM 性能

DOM 操作非常“昂贵”，避免频繁的 DOM 操作
对 DOM 查询做缓存
将频繁操作改为一次性操作

DOM 文档加载步骤：

1.解析 HTML 结构 2.加载外部的脚本和样式文件
3，解析并执行脚本代码 4.执行$function(){}内对应代码
【$(function() {}) 是$(document).ready(function()的简写。这个函数在 DOM 加载完毕之后执行。】
【window.onload：资源全部加载完才能执行，包括图片。DOMContentLoade ：DOM 渲染完成即可，图片可能尚未下载】 5.加载图片等二进制资源 6.页面加载完毕，执行 window.onload

jQuery 对象和 DOM 对象的相互转换

1. jQuery –> DOM
jQuery 提供了两种方法：[index]和 get(index)

jQuery 对象是一个类似数组的对象，可以通过[index]方法得到相应的 DOM 对象

另一种方法是 jQuery 本身提供的，通过 get(index)方法得到相应的 DOM 对象

2. DOM –> jQuery
只需用$()把 DOM 对象包起来即可
